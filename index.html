How I Developed This Page

What is Flask?

Flask is a web development framework that uses Python. It is used
for developing websites on a development server. I use flask to develop
my html website.

How do you use it?

Run 'app.py'. It will open a localhost server, and from there, you can update your
website (in my case, index.html), and develop it to your liking. It does not rely on a real
server, and the refresh time is instant. This makes developing a website very easy to use
and all you need is to have it installed to develop your html website.
Once finished, I used github to host it for the public.

Flask Installation

1. Assuming Python already installed.
2. Install Flask in command prompt using pip with command 'pip3 install Flask'
3. Created simple Python program that uses flask to render our html page (index.html).
4. Designed an index.html page, and styled with CSS.

Challenges Faced

- Flask follows specific filepaths (see flask documentation),
this was conflicting with github's hosting requirements.
- Ensuring the python app was launching the development server properly.

Solutions

- Followed Flask's documentation to manually set my 'template' and 'static' folders,
to comply with github's requirements
- Ensuring the correct 'language mode' I was running was the same as the one with flask installed.
This solution may vary depending on the ide you use (I'm using VSCode).
